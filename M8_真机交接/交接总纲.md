## 一、项目背景
- Synapse = Electron + React 19 + Redux Toolkit，打磨成 CC/Codex 式 agent harness。
- 仓库 JuryBu/Synapse，工程在 `synapse-app/`，分支 main。
- 双编译验证（**不要 vite build，会 OOM**）：
  ```
  cd synapse-app && node node_modules/typescript/bin/tsc -b   # renderer，期望 R=0
  node node_modules/typescript/bin/tsc -p tsconfig.electron.json   # electron，期望 E=0
  ```
- CDP 真机验证（dev 跑 localhost:5173 + electron --remote-debugging-port=9222）：
  `web-fetcher: desktop_connect_cdp(port 9222) → register_window → web_interact(action=evaluate, value=JS)`；store 在 `window.__SYNAPSE_STORE__.getState()`。
- 项目无 @modelcontextprotocol/sdk，MCP 是手搓 stdio JSON-RPC；Exa 走 HTTP Broker 127.0.0.1:14588。
