# Plan_3：功能实装与 UI 完善

> **背景**：Plan_2 完成了核心架构搭建（IPC、MCP 客户端、Agent 工具链、终端 UI），但实际用户验证发现大量功能仍为占位/伪实装状态。Plan_3 聚焦将所有 UI 组件从"看得见"升级为"用得了"。

## 总纲

Plan_3 包含以下子文档：
- **Plan_3_1_工作区管理.md** — 文件树操作实装 + 工作区生命周期
- **Plan_3_2_设置系统.md** — 通用/AI/对话/安全/Synopsis/MultiAI/插件 全面实装 + Tab 自适应
- **Plan_3_3_AI对话系统.md** — 真实 AI 通信 + 输入框/Token/连接状态/文件上传/模型切换/模式差异（8个问题）
- **Plan_3_4_终端与编辑器.md** — 终端实装 + 文件查看器加载 + 代码编辑器保存
- **Plan_3_5_知识概要与欢迎页.md** — Synopsis RAG 真实化 + 欢迎页功能闭环

## Stage 一览（预估）

| Stage | 范围 | 核心目标 |
|-------|------|---------|
| S1 | 工作区管理 | 文件树右键全实装 + 工作区打开/新建/删除 |
| S2 | 设置系统-通用 | 字号/语言/主题/强调色 全局生效 |
| S3 | 设置系统-壁纸 | 背景图选择/轮播/磨砂/透明度 真实渲染 |
| S4 | 设置系统-AI+对话 | API Key 管理 + 模型实时获取 + 参数生效 |
| S5 | 设置系统-其余Tab | 安全/Synopsis/MultiAI/插件/数据 实装 |
| S6 | AI 对话-核心通信 | 真实 streaming 对话 + 工具调用闭环 |
| S7 | AI 对话-增强 | 文件/图片上传 + 缩略图 + 模型列表切换 |
| S8 | AI 对话-模式差异 | Fast/Plan 模式行为区分 |
| S9 | 终端实装 | 真实命令执行（Web 模拟 / Electron node-pty）|
| S10 | 编辑器与查看器 | PDF/DOCX/PPTX 查看 + 代码编辑器增强 |
| S11 | 设置 Tab 响应式 | Tab 栏窄屏自适应 + 左右滚动按钮 |
| S12 | 整体联调 | 全功能端到端验证 |

## 问题来源

所有问题来自用户 2026-04-26 实际测试反馈，附截图佐证。
