# Plan_3_5：知识概要与欢迎页实装

## 问题描述

### P1：知识概要（Synopsis）全部虚假（第二批反馈-图一）
**现象**：
- 左侧「知识概要」面板显示 `2 已完成 / 3 待处理` — 但这是硬编码假数据
- 「第1章-绪论.pdf ✓ 5分片」「第2章-线性表.pdf ✓ 8分片」— 假的完成状态
- 第3-5章的「生成」按钮点击无响应
- 「全部生成」按钮也无功能

**根因**：Synopsis 面板（SynopsisPanel/KnowledgeBase 组件）的数据是从 demo 文件树映射出的静态 UI，synopsisEngine.ts 的 RAG 管线（文档解析→分片→向量化→索引）未真实接入前端。

### P2：欢迎页所有卡片无功能（第二批反馈-图二）
**现象**：
- 「打开工作区」卡片 — 点击无反应（应触发 Electron dialog 选择文件夹）
- 「新建课程」卡片 — 点击无反应（应创建空工作区）
- 「导入课件」卡片 — 点击无反应（应打开文件选择器导入 PDF/PPTX/DOCX）
- 「AI 助手」卡片 — 点击无反应（应跳转右侧对话面板）
- 「最近工作区 - 示例工作区」 — 点击无反应（应切换到该工作区）

**根因**：WelcomePage 组件的按钮 onClick 只注册了空函数或 console.log，未接入实际的工作区/文件系统/导航 API。

## 目标

### 知识概要真实化
1. 文件列表从实际工作区课件文件动态生成
2. 「生成」按钮 → 调用 synopsisEngine 解析文档 → 分片 → 存储
3. 进度状态实时反映（解析中/已完成/失败）
4. 已生成摘要可在对话中引用（RAG 检索）

### 欢迎页功能闭环
1. 「打开工作区」→ Electron `dialog.showOpenDialog` / Web `input[type=file]`
2. 「新建课程」→ 创建新工作区目录 + 初始化结构
3. 「导入课件」→ 文件选择 → 复制到工作区 → 刷新文件树
4. 「AI 助手」→ focus 到右侧对话面板
5. 最近工作区列表 → 点击切换 + 持久化历史
