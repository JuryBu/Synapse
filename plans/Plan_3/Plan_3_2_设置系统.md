# Plan_3_2：设置系统全面实装

## 问题描述

### P1：通用设置全部不生效（图二/图三）
**现象**：
- **字号**：滑块数值变化但全局文字大小没有响应
- **语言**：切换语言后 UI 文字不变（i18n 系统未接入）
- **主题**：深色/浅色切换后整体配色不变
- **强调色**：选择蓝色后，仍有大量紫色元素未跟随变化（替换不完全，图二可见设置tab图标、滑块等仍为紫色）

**根因**：Redux 中的设置状态变化了，但缺少「将状态值注入 CSS 变量 / 全局样式」的桥接层。

### P2：壁纸功能全面未实装（图二/图三）
**现象**：
- 背景图选择按钮可以选图片（已添加1张），但选择后没有任何视觉效果
- 轮播模式、磨砂度、面板透明度滑块数值可调，但视觉上无任何变化
- 清除按钮功能未知

**根因**：壁纸相关的 CSS 变量 / DOM 背景层未实现。

### P3：设置 Tab 栏截断无滚动（图二）
**现象**：当左侧面板宽度较窄时，「通用/AI/对话/安全/Synopsis/MultiAI/插件/数据/关于」Tab 栏会被截断，右侧的 Tab 完全不可见不可点击。
**期望**：Tab 栏在空间不足时应出现左右箭头按钮，或者可以横向滚动。

### P4：AI/对话/安全/Synopsis/MultiAI/插件 Tab 内容为空占位（图四）
**现象**：这些 Tab 内有部分 UI 控件，但功能全部是硬编码占位，没有真实逻辑：
- **AI Tab**：API Key 输入可以保存，但测试连接逻辑虽已修复，模型列表获取后不持久
- **对话 Tab**：历史压缩等设置无实际效果
- **安全 Tab**：所有选项仅 UI
- **Synopsis Tab**：RAG 引擎设置无实际效果
- **MultiAI Tab**：多 Agent 系统仅 UI
- **插件 Tab**：扩展市场完全空白
- **数据 Tab**：导出 JSON/清除缓存 按钮功能需验证

## 目标

### 通用设置生效
1. **字号** → CSS 变量 `--app-font-size` 全局注入，所有组件响应
2. **语言** → i18n 框架（react-i18next）接入，至少支持 中/英 双语
3. **主题** → 深色/浅色主题 CSS 变量集切换
4. **强调色** → CSS 变量 `--accent-color` + 所有使用紫色的地方改为引用此变量

### 壁纸系统
1. 背景图层 DOM 元素 + CSS 覆盖
2. 磨砂度 → `backdrop-filter: blur(Xpx)` 实时响应
3. 面板透明度 → 面板背景 `rgba` alpha 通道响应
4. 轮播 → 多图定时切换 + 过渡动画

### Tab 栏响应式
1. Tab 容器 `overflow-x: auto` + 自定义滚动箭头

### 设置 Tab 真实逻辑
按优先级渐进实装，部分可标注「计划中」。
